## `GET /api/clips/{id}`

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The clip ID |

### Example

```bash
curl https://api.scalereach.ai/api/clips/clip_abc123 \
  -H "Authorization: Bearer sr_live_YOUR_KEY"
```
